Aluguel de carros

O primeiro projeto de Python que eu fiz, E relacionado aluguel de carros, onde voçê faz um cadastro do seus dados, E o código faz um relatorio sobre quanto voçê ira pagar.

Abaixo tem o código do projeto!

                
                        from datetime import date
                        atual = date.today().year
                        dado = dict()
                        print(" ALUGUEL DE CARROS ")
                        atual = date.today().year
        
                        dado['nome'] = str(input("Digite seu nome completo: ")).strip().title()
                        nasc = int(input("Ano de nascimento: "))
                        dado['idade'] = atual - nasc
        
                        dado['cpf'] = int(input("Digite seu CPF: "))
                        dado['carro'] = str(input("Digite um modelo de carro: ")).strip().upper()
                        dado['dias'] = int(input("Duração com o veiculo: "))
                        dado['carteira'] = str(input("Tem CNH? [S/N] ")).strip().upper()
                        while dado['carteira'] not in 'SsNn':
                            dado['carteira'] = str(input("Por favor digite [S/N] se possui CNH: ")).strip().upper()
        
                        if dado["carteira"] in "Ss":
                            valor = 20 * dado["dias"]
                            print("\033[0;32mVocê pagará menos por ter CNH!\033[m")
                        else:
                            valor = 70 * dado["dias"]
                            print("\033[0;31mVocê pagará ao dobro por conta de não ter CNH!\033[m")
                        
                        print("="*30)
                        print("{:^30}".format("RELATORIO"))
                        print(f"Nome: {dado['nome']}")
                        print(f"Ano de nascimento: {nasc}")
                        print(f"Idade: {dado['idade']}")
                        print(f"CPF: {dado['cpf']}")
                        print(f"Veiculo: {dado['carro']}")
                        print(f"CNH: {dado['carteira']}")
                        print(f"Duração será de {dado['dias']} dias")
                        print(f"Valor R${valor}")

Contagem Regressiva

Neste projeto de Python que se chama Contagem Regressiva, Funciona basicamente voçê colocar quantos segundos quer que dure, E por aí o código vai contando até o fim.

Abaixo tem o código do projeto!

                
                    from time import sleep
                    print(" CONTAGEM REGRESSIVA ")
                    numero = int(input("Digite quantos segundos quer durar: "))
                    for a in range(1, numero+1):
                        print(a)
                        sleep(1)
                    print('FIM!')

                        nome = str(input("Digite um participante: ")).upper()

                    print("=" * 30)
                    print("{:^30}".format("LISTAGEM DE PARTICIPANTES"))
                    print("=" * 30)
                    print(f'{"ID"} {"JOGADOR":>20}')
                    for i, a in enumerate(ficha):
                        print("-" * 30)
                        print(f"{i+1}º {a:>20}")
                        print("-" * 30)
                    aa = random.choice(ficha)
                    print(f"O GANHADOR FOI {aa}")
